1. **Structure (20 points)**

   - Strict types
   - No deprecated APIs

3. **TYPO3 Patterns (25 points)**
   - Dependency injection
   - Event system usage
   - TCA configuration

4. **Testing (15 points)**

   - Functional tests
   - CI configuration

5. **Documentation (15 points)**
